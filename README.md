# aoc2024

## Usage

Run the project's tests contained in [Rich Comment Forms](https://github.com/hyperfiddle/rcf):

    $ clojure -X:test

Run a REPL

    $ clj -M:env/dev:repl

Build a jar

    $ clj -T:build jar
